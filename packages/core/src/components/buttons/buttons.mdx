# Button

To trigger an operation.

## When To Use

A button means an operation (or a series of operations). Clicking a button will trigger corresponding business logic.

We provide 3 types of button.

- Primary button
- Secondary button
- Outlined button

And 6 other properties additionally.

- none
- success
- warning
- danger
- special
- disabled

## Usage

```js
import { Button } from '@tpr/core';
```
