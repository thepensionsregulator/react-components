# Buttons

Buttons make common actions more obvious and help users more easily perform them. Buttons use labels and sometimes icons to communicate
the action that will occur when the user touches them.

## Properties

<Props of={Button} />

## Usage

```js
import { Button } from '@tpr/core';
```
